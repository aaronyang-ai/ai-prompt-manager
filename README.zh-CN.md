# AI提示词管理器

[English](README.md) | 简体中文

一个强大的Chrome浏览器扩展，支持在ChatGPT、Claude、Gemini、Grok、Perplexity、DeepSeek、豆包、千问等8个主流AI平台高效管理和快速插入预设提示词。

## ✨ 功能特性

### 🎯 核心功能
- **智能侧边菜单**：悬浮触发按钮，展开完整的提示词管理界面
- **一键插入**：点击提示词直接插入到AI输入框，光标自动定位到文本末尾
- **分类管理**：支持自定义分类，井然有序地组织提示词
- **快速搜索**：实时搜索功能，快速定位所需提示词
- **分类折叠**：支持分类展开/折叠，便于浏览和管理

### 💡 高级特性
- **多平台支持**：完美适配ChatGPT、Claude、Gemini、Grok、Perplexity、DeepSeek、豆包、千问等8个主流AI平台
- **拖拽定位**：触发按钮支持自由拖拽，个性化界面布局
- **智能按钮管理**：智能收起机制，支持多种触发条件自动收缩
- **精准光标定位**：插入文本后光标自动定位到合适位置，便于继续编辑
- **本地存储**：所有数据本地保存，隐私安全有保障
- **导入导出**：支持备份和迁移提示词数据

### 🎨 用户体验
- **现代化界面**：精美的UI设计，符合现代审美
- **流畅动画**：平滑的过渡效果和交互反馈
- **智能交互**：自动收起、延时收缩、点击外部关闭等智能化交互
- **响应式设计**：适配不同屏幕尺寸
- **中文优化**：针对中文用户习惯优化

## 🚀 安装方法

### 开发者模式安装
1. 克隆或下载本仓库到本地
```bash
git clone https://github.com/aaronyang-ai/ai-prompt-manager.git
```
2. 打开Chrome浏览器，进入扩展管理页面：`chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"按钮
5. 选择本项目文件夹

## 📖 使用指南

### 首次使用
1. 安装扩展后，访问 [ChatGPT](https://chat.openai.com/) 或 [Claude](https://claude.ai/)
2. 在页面右侧会出现悬浮的提示词触发按钮（默认收缩状态）
3. 鼠标悬停按钮自动展开，点击打开侧边菜单
4. 按钮支持智能收起：用完后自动收缩，点击空白处收缩，或3秒无操作自动收缩

### 管理提示词
- **添加提示词**：在弹窗界面或侧边菜单中点击"添加"按钮
- **分类整理**：为提示词设置分类标签，便于组织管理
- **编辑修改**：点击提示词卡片的编辑按钮进行修改
- **快速搜索**：使用搜索框快速定位特定提示词
- **导入导出**：支持JSON格式的数据备份和迁移

### 使用提示词
1. 在ChatGPT/Claude的输入框中，点击提示词触发按钮
2. 从侧边菜单中选择需要的提示词
3. 提示词会自动插入到输入框中，光标定位到文本末尾
4. 可以直接继续输入补充内容，或直接发送
5. 使用完成后，按钮和菜单会智能收起，保持界面整洁

- **样式技术**：CSS3 + 现代化动画
- **交互技术**：DOM Selection API、事件委托、状态管理
- **性能优化**：防抖技术、内存管理、硬件加速动画
- **构建工具**：原生开发，无需复杂构建流程
